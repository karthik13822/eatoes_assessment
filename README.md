# Restaurant Admin Dashboard (Eatoes Intern Assessment)

This project is a full-stack Restaurant Admin Dashboard built as part of the Eatoes Intern Technical Assessment.  
It allows restaurant owners to manage menu items, track orders, and update availability in real time.

The application is built using the MERN stack (MongoDB, Express, React, Node.js) and follows a clean client-server architecture.

---

## Features

### Menu Management
- Add new menu items
- View all menu items
- Search menu items by name or ingredients
- Toggle availability status
- Delete menu items

### Orders Management
- View all customer orders
- Update order status (Pending, Preparing, Ready, Delivered, Cancelled)
- View order details

### General
- RESTful API architecture
- MongoDB Atlas cloud database
- Environment variable configuration
- Responsive and clean UI
- Error handling and validation
